WIPO IP Ontology (draft): Description back to ToC

Draft schema (TBox only) for comparing national IP laws against WIPO treaty minimum standards. Built following Palmirani's MeLOn methodology, reusing FRBR (frbr:) for the document layer and LegalRuleML's metamodel (lrmlmm:) for the statement layer. Covers CQ1-CQ9 (national side); the treaty side (CQ10) is in the separate treaty_draft.ttl, marked as discussion material. Worked examples (Ireland, India, Kenya) that validate this schema against real statute text live in examples.ttl, kept separate on purpose: this file stays a stable, roughly constant-size schema no matter how many jurisdictions the pipeline eventually covers (up to ~200), while worked examples grow one file per batch of countries. Split 2026-07-23.
